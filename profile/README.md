The PCRE4J project's goal is to bring the power of the [PCRE](https://www.pcre.org) library to Java.

PCRE4J provides **100% coverage of the PCRE2 API**, giving you access to every feature of the PCRE2 library from Java.

## Why PCRE4J?

- **Richer regex syntax** — atomic groups, `\K`, recursive patterns, callouts, DFA matching
- **JIT compilation** — native machine code for significant speedups
- **Built-in ReDoS protection** — configurable match, depth, and heap limits
- **Cross-system consistency** — match behavior with Nginx, PHP, R, grep -P, and more
- **Drop-in `java.util.regex` API** — switch engines without rewriting code
- **Full PCRE2 API access** — 100% of the C API exposed to Java
